== Introduction

The OPAL Tool-chain Builder (OTB) is a collection of recipes to build all
required software to compile _OPAL_.

The documentation how to setup and run the recipes is in the 
https://gitlab.psi.ch/OPAL/src/-/wikis/For-Developers/Compile-OPAL#user-content-build-the-opal-tool-chain-from-scratch[OPAL Wiki].
